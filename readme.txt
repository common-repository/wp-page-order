Simple Page Ordering - New Version Support Latest WP Version 4.4 . Order your pages and other hierarchical post types with simple drag and drop right from the standard page list.

1. Download plugin zip file to your computer and extract in a folder
1. Upload `wp-page-order` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
